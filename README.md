# Stella ERP - Sistema de Gestión para Joyería Artesanal

##  Objetivo del Proyecto

Desarrollar un sistema ERP web modular e inteligente para digitalizar y centralizar
las operaciones administrativas, comerciales y de atención al cliente de Stella Joyería.

##  Integrantes

- Ana Rosa Mamolejo Cervantes
- Estela Barajas Lozano

##  Tecnologías

- **Frontend:** Next.js 14, React 18, Tailwind CSS
- **Backend:** Supabase, PostgreSQL, Node.js
- **IA/ML:** Python, TensorFlow, Gemini API
- **DevOps:** Docker, GitHub Actions, Vercel

## Instrucciones para Ejecutar

1. Clonar repositorio: `git clone  https://github.com/Estela386/Stella-ERP.git`
2. Instalar dependencias: `npm install`
3. Configurar variables: `cp .env.example .env.local`
4. Ejecutar: `npm run dev`
5. Acceder: http://localhost:3000
